~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
To run application:
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

1.0 Open a terminal\console

2.1 To run the application with the default locale (en-AU : english), type:

	./gradlew run


2.2 To run the application with the alternative locale (fr-FR : french), type:

	./gradlew run --args='--locale=fr-FR'


~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
Where to find plugins/scripts:
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

The plugins are found in the following locations:

	DatePlugin : SEC_Assignment2\proj_pluginDate\bin\src\main\java\texteditor\DatePlugIn.class

	FindPlugin : SEC_Assignment2\proj_pluginFind\bin\src\main\java\texteditor\FindPlugIn.class


The emoji script is found in the following location:

	emoji :  SEC_Assignment2\emoji.py
